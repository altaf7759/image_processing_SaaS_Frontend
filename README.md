# Image Processing SaaS Frontend

A modern React frontend for a production-grade Image Processing SaaS platform. The application provides real-time image processing updates, authentication workflows, batch upload management, and responsive dashboard experiences.

Built with React, React Router, Context API, and modern frontend architecture practices.

---

# Features

* Responsive React-based UI
* Protected routing with React Router
* Global state management using Context API
* Real-time processing updates via SSE
* Authentication and session handling
* Batch image upload workflow
* Live job progress tracking
* Clean and scalable component architecture

---
